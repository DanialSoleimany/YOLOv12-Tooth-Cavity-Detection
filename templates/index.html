A modern web dashboard for automatic dental cavity detection using deep learning and YOLOv12.

Features
  • Upload dental X-ray images and view instant predictions.
  • Automatic annotation of teeth and cavity detection.
  • Probability bar chart for each annotated tooth prediction.
  • Modern, responsive UI with sidebar navigation.
  • Easy-to-use dashboard for dentists and researchers.
How It Works
  1. Go to the Prediction page.
  2. Upload a dental X-ray image.
  3. View the annotated image and prediction results.
  4. See the probability of cavity for each tooth in the bar chart.
About

This project uses YOLOv12 for object detection on dental X-rays, trained on the Cavity Dataset from Roboflow Universe. It is designed for dental professionals and researchers to easily analyze cavity risks in X-ray images.

For more information, visit the dataset page or project repository.
